# Documentation

This folder contains all project documentation, guides, and design documents.

## Structure

- **Design Documents**: Original design specifications
  - `asset_design_doc.txt` - Asset page design specifications
  - `workflow_design.txt` - Workflow page design specifications

- **Setup Guides**: Step-by-step guides for setting up various features
  - `IMPORT-WORKFLOWS-GUIDE.md` - How to import workflows from JSON
  - `UPLOAD-WORKFLOW-IMAGES-GUIDE.md` - How to upload workflow images to Supabase
  - `MANUAL-UPLOAD-INSTRUCTIONS.md` - Manual upload instructions for workflow images
  - `VERIFY-UPLOAD-CHECKLIST.md` - Checklist for verifying image uploads
  - `README-ONTOLOGY.md` - Documentation for the ontology initialization system
  - `WORKFLOWS-SETUP.md` - Complete workflow setup guide
  - `UPLOAD-WORKFLOW-IMAGES.md` - Workflow image upload guide
  - `setup-storage.md` - Supabase storage setup instructions
  - `import-functions.md` - Function import guide
